Whitespace in HTML


    <p id="noWhitespace">Dogs are silly.</p>

    <p id="whitespace">Dogs
        are
            silly.</p>

Whitespace in HTML is ignored by the browser. This means that you can add extra spaces, tabs, and line breaks to your HTML code to make it more readable without affecting how it is displayed in the browser. It can also get weird with js and innerHTML
