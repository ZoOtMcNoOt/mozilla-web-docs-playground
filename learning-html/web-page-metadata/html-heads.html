Title

The <title> element specifies the title of the HTML document, which is displayed in the browser's title bar or tab.

the <meta> element

The metadata is the data that describes the data and html in the doc.

The <meta> element is used to specify the metadata for an HTML document, such as the character set, author, and viewport settings.

Other types of metadata

There are many other types of metadata that can be included in the head of an HTML document, such as links to external stylesheets, scripts, and favicons.

For example, the <link rel="stylesheet" href="styles.css"> element links to an external CSS file that styles the HTML document.

Favicons are small icons that represent a website in the browser's address bar or tab. They can be specified using the <link rel="icon" href="favicon.ico"> element.

Another example is Open Graph Data which is a metadata protocol that Facebook invented to give a richer metadata for websites.

Open Graph Data is used to specify how a webpage should be displayed when shared on social media platforms like Facebook, Twitter, and LinkedIn.

For example, the <meta property="og:title" content="My Webpage"> element specifies the title of the webpage when shared on social media.

Other Open Graph properties include <meta property="og:description" content="A brief description of my webpage"> and <meta property="og:image" content="image.jpg"> to specify an image to be displayed when the webpage is shared.

Adding Custom Icons to your site

To add to site design you can add references to custom icons in your metadata which will be displayed in certain contexts, such as browser tabs or bookmarks.

For example, the <link rel="icon" href="favicon.ico"> element specifies a custom icon for the website. which is 16px by 16px. It is saved in .ico format but can also support other formats like PNG, SVG and GIF.

Applying CSS and JS to HTML

You can apply CSS styles to your HTML document by linking to an external stylesheet or by using inline styles.

To link to an external CSS file, you can use the <link> element in the head of your HTML document:

<link rel="stylesheet" href="styles.css">

To apply inline styles, you can use the style attribute on individual HTML elements:

<h1 style="color: blue;">Hello, World!</h1>

JavaScript can be added to your HTML document using the <script> element. You can link to an external JavaScript file or include inline scripts:

<script src="script.js"></script>

Inline scripts can be added directly within the <script> tags:

<script>
    alert("Hello, World!");
</script>

The script element is used to define client-side JavaScript code in an HTML document. use defer boolean attribute to tell the browser to load the javascript after the page has finished parsing the html so there are no errors due to js trying to access an html element that doesnt exist on the page.

Primary Language

The <meta http-equiv="Content-Language" content="en"> element specifies the primary language of the HTML document.
